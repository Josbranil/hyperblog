Hyperblog Tu blog de cabecera

Este es mi blog de práctica del curso de Git y GitHub

Con Git podemos usar diferentes ramas para realizar los cambios que queramos a los archivos de nuestro repositorio trabajando de forma paralela a nuestra rama principal para no modificar los archivos originales hasta que queramos implementar dichos cambios, así cuando estemos listos podemos fusionar esas ramas con la rama principal

Los blogs son la mejor forma de compartir información e ideas. Mucho más que ir a conferencias o salir en Youtube. Excepto si eres un rockstar. Pero estadísticamente no lo eres.... por ahora.
